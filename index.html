Mobala FX

Web App

Mobala FX is an innovative money exchange platform designed to facilitate seamless transactions between users and money changers. By enabling direct interactions without the need for traditional banks, people can experience the freedom of exchanging currencies conveniently.

As a full-stack developer, I played a pivotal role in leading both frontend and backend development, overseeing the project's progress to ensure its successful realization. This cutting-edge project utilizes a diverse technology stack, incorporating JavaScript, HTML, CSS, C#, and the React framework for the frontend, along with Dotnet Core for the backend.

One of the prominent features of Mobala FX is its end-to-end encrypted chat system, ensuring secure communication during money exchange transactions.

Throughout the development process, we encountered various challenges, particularly in establishing a reliable peer-to-peer connection between users. The implementation of real-time notifications, messages, and order invitations posed a novel experience for me as it was my first project in this domain.

To surmount these obstacles, we adopted the ASP.NET SignalR library, which facilitated seamless peer-to-peer connections using hubs. This decision proved instrumental in achieving an effective and efficient solution for the platform.

Adamjee Academy

Mobile App

The Adamjee Academy app is a cross-platform application that serves as a vital communication tool between the academy, enrolled students, and their parents. Its primary objective is to keep parents informed about their children's attendance, academic results, class schedules, and whether their child was present during a particular class session.

As a dedicated backend developer, I played a pivotal role in spearheading the backend development process. My responsibilities encompassed creating APIs, designing the database structure, and proficiently querying data. Throughout the development cycle, I diligently monitored the project's progression to ensure its successful realization.

This cutting-edge project leverages a versatile technology stack, incorporating TypeScript, HTML, CSS, C#, and the React Native framework for the frontend, while relying on Dotnet Core for robust backend support and Firebase for realtime messaging service.

Throughout the development journey, we encountered several challenges, notably in implementing real-time notifications for parents. Enabling parents to receive timely updates about their children's classes and absences proved to be a novel experience, especially as it was my inaugural project in this domain.

To conquer this obstacle, we strategically adopted the Firebase Realtime Message Service, which allowed us to efficiently store device information and promptly send relevant updates to users. By leveraging this solution, we achieved seamless real-time notifications, enhancing the overall effectiveness of the platform.
